# devradar
Finding nearby devs

## Technologies Used
 - Javascript
 - Node.js
 - ReactJs
 - React Native
 - MongoDB
  
## Dependencies
 - axios 0.19.1
 - cors 2.8.5
 - express 4.17.1
 - mongoose 5.8.7
 - nodemon 2.0.2
 - socket.io 2.3.0
  
## How to get the project
```sh
git clone https://github.com/jonasdiego21/devradar.git
```

## How to install the dependencies
```sh
cd devradar
npm install 
```

## How to start the project
```sh
npm dev
```

> In this project, npm build, test and start were not added
